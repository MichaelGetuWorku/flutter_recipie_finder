# recipie_finder

A Recipe finder and store Flutter project.
Recipes are found from <a href="https://www.edamam.com/">Visit Edamam</a>

## Packages used

<ul>
<li>Catched network images</li>
<li>flutter Slidable </li>
<li>flutter Svg </li>
<li>Shared preferences - to store the bookmarked recepies</li>
<li>Json Annotation</li>
<li>HTTP - to send http requests</li>
<li>Chopper</li>
<li>Logging</li>
</ul>
